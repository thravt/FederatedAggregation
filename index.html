Project Overview

This research project explores the performance of different federated aggregation methods—FedAvg, FedLAMA, and FedDist—on a variety of deep learning architectures and datasets. It aims to answer how these methods perform across vision, tabular, and text domains using a consistent evaluation pipeline.

Technical Summary

  • FedAvg: Standard model averaging based on local client weights
  • FedLAMA: Layer-wise adaptive aggregation that accounts for gradient variability
  • FedDist: Output-level distillation using a public dataset for soft-label aggregation

Each method was tested on:

  • CNN model on CIFAR-10 (image classification)
  • MLP model on MIMIC-III (clinical tabular data)
  • DistilBERT on Sentiment140 (text sentiment classification)

Repository

Explore the full codebase and experiment results here: https://github.com/thravt/FederatedAggregation

Key Takeaways

  • FedLAMA generally achieved more stable convergence in smaller models
  • FedDist proved effective when a small public dataset was available for distillation
  • FedAvg remained competitive in simpler tasks with balanced data
